设计模式：Process() 单采样处理

DaisySP 的每个 DSP 模块都遵循统一接口：Init() 初始化 + Process() 处理单个采样。这个设计简洁优雅，便于组合。

DaisySP 的统一 DSP 接口模式
// DaisySP 的典型 DSP 模块结构
class Oscillator {
public:
    // ① 初始化 — 设置采样率、波形类型
    void Init(float sample_rate) {
        sr_ = sample_rate;
        phase_ = 0.0f;
        freq_ = 440.0f;
        // 预计算相位增量
        phase_increment_ = freq_ / sr_;
    }

    // ② 设置参数 — 运行时可调
    void SetFreq(float freq) {
        freq_ = freq;
        phase_increment_ = freq_ / sr_;
    }

    // ③ Process — 处理一个采样，返回一个采样
    float Process() {
        float out;
        // 多项式逼近正弦波（比查表更快）
        out = sinf(phase_ * 2.0f * PI_F);

        // 更新相位
        phase_ += phase_increment_;
        // 相位回绕到 [0, 1)
        if (phase_ >= 1.0f) phase_ -= 1.0f;

        return out;
    }

private:
    float sr_;
    float phase_;
    float freq_;
    float phase_increment_;
};

// ④ 在音频回调中使用 — 模块组合
void AudioCallback(float* in, float* out, size_t size) {
    for (size_t i = 0; i < size; i += 2) {
        // 振荡器 → 滤波器 → 输出
        float sig = osc_.Process();          // 生成波形
        sig = filter_.Process(sig);          // 滤波
        sig *= 0.5f;                         // 音量

        out[i]     = sig;  // 左
        out[i + 1] = sig;  // 右
    }
}
① Init() 在启动时调用一次。传入采样率，预计算常量。这是嵌入式 DSP 的标准模式 — 预计算越多，Process() 越快。
③ Process() 返回一个 float。每个采样调用一次。在 48kHz 下，每秒调用 48000 次。必须尽量快 — 使用 FPU 的单周期浮点乘法。
④ 模块组合像搭积木：Oscillator → Filter → 效果器 → 输出。每个模块独立、无状态依赖（除了内部 phase/state）。这种设计在嵌入式音频中非常流行。

源码走读：Moog 滤波器

Moog 梯形滤波器是经典模拟合成器滤波器的数字实现。DaisySP 的实现展示了如何在嵌入式 DSP 中平衡精度和性能。

Source/Filters/moogladder.cpp — Moog 滤波器核心
float MoogLadder::Process(float in) {
    // ① 4 级梯形滤波器
    float freq = freq_;  // 截止频率
    float res  = res_;   // 谐振 (0~1)

    // ② 频率归一化到 [0, 1] 范围
    float f = 2.0f * freq / sample_rate_;
    // 限制到 Nyquist 频率以下
    f = f > 1.0f ? 1.0f : (f < 0.0f ? 0.0f : f);

    // ③ 计算滤波器系数
    // 使用 Stilson/Smith 的改进梯形拓扑
    float k = 3.6f * res;  // 谐振增益
    float p = f * (1.8f - 0.8f * f);  // 频率映射

    // ④ 四级低通滤波级联
    float t1 = (1.0f - p) * 1.386249f;
    float t2 = 12.0f + t1 * t1;

    // 级联处理
    float x = in - k * d4_;             // 输入减去反馈
    float a = d1_ + p * x;
    d1_ = a - x * (p - 1.0f) / t2;     // ⑤ 第一级状态更新

    float b = d2_ + p * d1_;
    d2_ = b - d1_ * (p - 1.0f) / t2;   // 第二级

    float c = d3_ + p * d2_;
    d3_ = c - d2_ * (p - 1.0f) / t2;   // 第三级

    float d = d4_ + p * d3_;
    d4_ = d - d3_ * (p - 1.0f) / t2;   // ⑥ 第四级

    // ⑦ 输出 = 四级级联结果
    return d;
}

void MoogLadder::Init(float sample_rate) {
    sample_rate_ = sample_rate;
    freq_ = 1000.0f;    // 默认截止频率 1kHz
    res_ = 0.0f;        // 默认无谐振
    d1_ = d2_ = d3_ = d4_ = 0.0f;  // 清零状态
}
① 梯形拓扑：Moog 合成器的标志性声音来自 4 级低通级联 + 谐振反馈。这是模拟电路的数字仿真。
②③ 频率归一化：截止频率从 Hz 转换为 [0,1] 归一化频率。f=1 对应 Nyquist 频率（采样率/2）。这是所有数字滤波器的标准做法。
⑤⑥ 四级状态 (d1-d4)：每级保存一个内部状态（前一采样输出）。4 个 float = 16 bytes RAM — 非常轻量。这就是嵌入式 DSP 和桌面 DSP 的区别。
⑦ 无除法运算：t2 预计算在循环外。嵌入式 DSP 中除法非常慢（12-20 周期），乘法只需 1 周期。DaisySP 处处体现了这种优化思维。
